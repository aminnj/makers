### setup
`mkdir -p plots data`


Need python2.7 with matplotlib, numpy, scipy, scikit-learn
```
brew install pip
pip install matplotlib
pip install numpy
pip install scipy
pip install scikit-learn
```
Install TA-lib (https://github.com/mrjbq7/ta-lib):
```
brew install ta-lib
pip install TA-lib
```
may need to sudo them
